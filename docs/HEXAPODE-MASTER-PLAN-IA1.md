# Hexápode — Plano Mestre da IA 1

## Meta

Construir uma biblioteca gráfica médica de 600 ativos reutilizáveis, com identidade visual única para todo o ecossistema Hexápode.

## Arquitetura

A matriz original possui 200 conceitos. Cada conceito terá três versões:

- `G` — canônica: ativos 001–200;
- `A` — aprofundada: ativos 201–400;
- `X` — aplicada: ativos 401–600.

Correspondência direta:

```text
conceito 001 → HEX-G001, HEX-A201, HEX-X401
conceito 200 → HEX-G200, HEX-A400, HEX-X600
```

## Produção

- 60 ciclos;
- 10 imagens por ciclo;
- cada ciclo autocontido, revisável e versionável;
- nenhum ativo concluído pode ser repetido;
- o manifesto é a fonte de verdade do estado de produção.

Os ativos 201–400 aprofundam os mesmos conceitos com maior granularidade anatômica, fisiológica, técnica ou clínica.

## Camada aplicada

Os ativos 401–600 mostram os mesmos conceitos em composições didáticas, comparativas, decisórias, procedimentais ou interativas.
